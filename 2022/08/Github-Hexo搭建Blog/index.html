更换主题

  1. 找到想要的主题，我用的是https://github.com/lizehongss/hexo-theme-zhl
  2. clone到themes目录下
  3. 在 _config.yml文件中修改theme: hexo-theme-zhl
  4. 执行hexo g & hexo s
  5. hexo deploy部署

写文章

  1. hexo new 文章名
  2. 在对应md文件中编辑

修改图标/链接
